"For a **long-term Data Cloud Consultant / Data Platform Architect** career, avoid collecting every certification. Focus on a **minimum viable stack** that proves expertise across cloud, data engineering, lakehouse, warehousing, governance, and architecture.

## Target End-State Profile

| Area                   | Primary Certification                                 |
| ---------------------- | ----------------------------------------------------- |
| Multi-Cloud Foundation | AWS + Azure + GCP                                     |
| Data Engineering       | GCP Data Engineer                                     |
| Lakehouse              | Databricks Data Engineer                              |
| Cloud Data Warehouse   | Snowflake SnowPro                                     |
| Architecture           | GCP Architect or AWS Solutions Architect Professional |
| Governance             | CDMP (optional later)                                 |
| Analytics              | Power BI / Fabric (optional)                          |
