
"We are looking for a Senior Data Engineer to design and build scalable data pipelines and analytics platforms using modern data engineering tools. The role involves data ingestion, transformation, data modeling, and visualization using cloud-based platforms such as Databricks and Power BI.


The ideal candidate will help build reliable data pipelines, enterprise-grade data models, and analytics dashboards to support data-driven decision making.


Key Responsibilities:


Data Pipeline Development


Design and develop scalable data ingestion pipelines from APIs, databases, cloud storage, and streaming sources.

Build automated pipelines to ingest structured and unstructured data.

Implement batch and near real-time data processing frameworks.

Ensure performance, reliability, and scalability of data pipelines.

Data Transformation & Processing


Develop ETL/ELT pipelines to clean, transform, and enrich datasets.

Build reusable data transformation workflows using distributed data processing frameworks.

Optimize data processing for performance and scalability.

Data Modeling & Data Warehouse


Design analytics-ready data models and dimensional schemas (Star/Snowflake).

Develop Bronze, Silver, and Gold layers in a lakehouse architecture.

Ensure data quality, consistency, and governance.

Analytics & Visualization


Develop Power BI dashboards and reports to support business insights.

Build semantic models and optimize queries for large-scale reporting.

Data Governance & Platform Optimization


Implement data quality checks and monitoring frameworks.

Define data governance policies, metadata management, and data lineage.

Improve performance, reliability, and cost efficiency of the data platform.

Collaboration


Work closely with data scientists, analysts, and business teams to understand requirements.

Translate business needs into scalable data engineering solutions.

Provide technical guidance and mentorship to junior engineers.
